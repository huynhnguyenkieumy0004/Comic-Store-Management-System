# Comic Store Management System

This is an academic web project developed for learning purposes.

## Features
- Comic management (add, edit, delete)
- Category management
- Product listing and search
- Basic CRUD operations

## Technologies
- ASP.NET MVC
- SQL Server
- HTML, CSS, JavaScript
